# 漢字フラッシュカード (Kanji Flashcards)

自分で作成した問題セットを使って、効率的に漢字の「読み」と「書き」を学習できるWebアプリケーションです。
シンプルで直感的な操作性、優れた視認性、そして軽快な動作を特徴としています。

**[実際のアプリを開く](https://nari0518.github.io/Kanji-Flashcards/)**

## 主な機能

**クイズモード**
- **読みモード**: キーボードから読みを入力して正誤判定（自動フォーカス対応でサクサク進めます）。または、答えを見て自分で判定する「自己判定」モードも搭載。
- **書きモード（自己判定）**: 画面に表示される問題（例：明日から（学校）が始まる。）を見て紙に書き、答えを表示して自分で「正解」「不正解」を判定します。

**問題セット管理**
- 画面上から簡単に新しい問題セットを作成・編集・削除できます。
- **CSVファイルでの入出力**: エクセルなどで作ったリスト（漢字, 読み, 例文）を一括でインポートしたり、現在のセットをバックアップ（エクスポート）したりできます。
- **フォルダからの自動読み込み**: `public/sets/` フォルダの中にCSVファイルを置いておくだけで、アプリ起動時に自動で読み込まれ、ファイル名がそのままセット名として登録されます。

**学習支援機能**
- **シャッフル機能**: 出題順をランダムにして、暗記の偏りを防ぐことができます。
- **学習履歴の記録**: いつ、どのセットを、どのモードで学習し、何点だったか（間違えた問題はどれか）がブラウザに記録されます。

## 独自のCSV問題セットを追加する方法

### 自動読み込み（リポジトリへの配置）

`public/sets/` フォルダにCSVファイルを置くだけで、アプリ起動時に自動で読み込まれます。

1. `public/sets/` フォルダを開く
2. 以下の形式のCSVファイルを保存
   - ファイル名例: `小学1年生の漢字.csv`
3. CSVの構成
   - 1行目: ヘッダー（見出し）
   - 2行目以降: データ

**CSVの記述例**

```csv
漢字,読み,例文
学校,がっこう,明日から（学校）が始まる。
勉強,べんきょう,毎日（勉強）をする習慣をつける。
友達,ともだち,（友達）と一緒に遊ぶ。
```

**注意事項**
- 例文の中で問題として出題したい（隠したい・読ませたい）部分を `（ ）` で囲んでください
- ファイルを配置後、アプリをリロードするか、GitHubにプッシュして自動デプロイを待ってください
- ファイル名（.csvを除いた部分）がそのままセット名として登録されます

### UI から直接追加

アプリの「セット管理」画面からCSVファイルをインポートすることもできます。

## 開発（ローカル起動）とデプロイ

このアプリは **React** と **Vite** で構築されています。

### ローカルでの動かし方

```bash
# パッケージのインストール
npm install

# 開発サーバーの立ち上げ（ブラウザで http://localhost:5173 を開く）
npm run dev

# 手動でビルドする場合
npm run build

# ビルド結果をプレビュー
npm run preview
```

### GitHub Pagesへの自動デプロイ

このリポジトリは GitHub Actions を使用して、`main` ブランチに変更がプッシュされると自動的に GitHub Pages に公開されるように設定済みです（`.github/workflows/deploy.yml`）。

公開用URL: `https://nari0518.github.io/Kanji-Flashcards/`

## 使用技術

- **React** (Hooks: useState, useEffect, useRef)
- **Vite** - 高速なビルドツール
- **Vanilla CSS** (カスタムプロパティ、CSSアニメーション)
- **LocalStorage** - ブラウザでのデータ永続化
- **GitHub Actions** - CI/CD自動デプロイ

## データフロー

### 問題セットの読み込み経路

1. **ローカルストレージから読み込み**: ユーザーが以前保存した問題セット
2. **静的問題セット（public/sets）から読み込み**: 
   - `public/sets/` フォルダに配置されたCSVファイル
   - ビルド時に `vite-plugin-generate-sets.js` が自動的に `index.json` マニフェストを生成
   - アプリ起動時に自動でマニフェストを読み込み、CSVをパース

### データの永続化

- **問題セット**: LocalStorageに JSON形式で保存
- **学習履歴**: LocalStorageに JSON形式で保存
- ブラウザを閉じてもデータは保持されます
