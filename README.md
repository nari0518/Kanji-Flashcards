# 漢字フラッシュカード (Kanji Flashcards)

_以下はすべてAIによる記述です。誤りが含まれている可能性があります。_

自分で作成した問題セットを使って、効率的に漢字の「読み」と「書き」を学習できるWebアプリケーションです。
シンプルで直感的な操作性、高い視認性（ダークモード＆ハイコントラスト）、そして軽快な動作を特徴としています。
R7現国の学年末対応

🌐 **[実際のアプリを開く](https://nari0518.github.io/Kanji-Flashcards/)**

## ✨ 主な機能

- **「読み」と「書き」の2モード対応**
  - **読みモード**: キーボードから読みを入力して正誤判定（自動フォーカス対応でサクサク進めます）。または、答えを見て自分で判定する「自己判定」モードも搭載。
  - **書きモード（自己判定）**: 画面に表示される問題（例：明日から（がっこう）が始まる。）を見て紙に書き、答えを表示して自分で「正解」「不正解」を判定します。
- **問題セットの自由な管理**
  - 画面上から簡単に新しい問題セットを作成・編集・削除できます。
  - **CSVファイルでの入出力**: エクセルなどで作ったリスト（漢字, 読み, 例文）を一括でインポートしたり、現在のセットをバックアップ（エクスポート）したりできます。
- **フォルダからの自動読み込み（新機能！）**
  - `public/sets/` フォルダの中にCSVファイルを置いておくだけで、アプリ起動時に自動で読み込まれ、ファイル名がそのままセット名として登録されます。
- **シャッフル機能**
  - 出題順をランダムにして、暗記の偏りを防ぐことができます。
- **学習履歴の記録**
  - いつ、どのセットを、どのモードで学習し、何点だったか（間違えた問題はどれか）がブラウザに記録されます。

## 📁 独自のCSV問題セットを追加する方法（自動読み込み）

ローカル環境や、GitHubのリポジトリ上から直接問題を追加することができます。

1. `public/sets/` フォルダを開きます。
2. その中に、以下のような形式のCSVファイル（カンマ区切り）を保存します。
   - 例: `小学1年生の漢字.csv`
3. CSVの中身は、1行目をヘッダー（見出し）、2行目以降をデータとして構成します。

**【CSVの記述例】**
```csv
漢字,読み,例文
学校,がっこう,明日から（学校）が始まる。
勉強,べんきょう,毎日（勉強）をする習慣をつける。
友達,ともだち,（友達）と一緒に遊ぶ。
```
※ 例文の中の、問題として出題したい（隠したい・読ませたい）部分を `（ ）` で囲んでください。

ファイルを置いた状態でアプリをリロード（またはGitHub Actionsによって自動公開）されると、 `小学1年生の漢字` という名前のセットが自動的に追加されます！

## 🚀 開発（ローカル起動）とデプロイ

このアプリは **React** と **Vite** で構築されています。

### ローカルでの動かし方
```bash
# パッケージのインストール
npm install

# 開発サーバーの立ち上げ（ブラウザで http://localhost:5173 を開く）
npm run dev

# 手動でビルドする場合
npm run build
```

### GitHub Pagesへの自動デプロイ
このリポジトリは GitHub Actions を使用して、`main` ブランチに変更がプッシュ（アップロード）されると自動的に GitHub Pages に公開されるように設定済みです（`.github/workflows/deploy.yml`）。
公開用URL: `https://[ユーザー名].github.io/Kanji-Flashcards/`
